version control

  • keeps track of text files
  • most common: Git
  • one repository per project
  • central project description

$\rightarrow$ only sensible if used consistently

Github/Bitbucket/Gitlab for collaboration

virtualization

  • preserves executables
  • avoids dependency conflicts
  • general solutions: Apptainer, Docker, ...
  • Python: Venv, Conda
  • Commercial solutions (Matlab, ...)

$\rightarrow$ greatly beneficial for collaboration

data archiving

  • ideally with backup (e.g. online)
  • raw data, processed data, containers
  • hard disk(s), RAID system
  • blob storage (temporary exchange)
  • long-term storage (final data, DOI)

hard disk(s)$\ge$ blob storage $\ge$ long-term storage

A practical example

Overleaf = Git+Docker+online storage for collaborative writing of Latex documents
